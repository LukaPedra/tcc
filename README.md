# Blackjack RL Agents

This project trains and compares Q-Learning and Monte Carlo agents on the Gymnasium Blackjack-v1 environment.

## Files
- `main.py`: Main script for training and plotting results.
- `Agents.py`: Contains agent classes (Q-Learning, Monte Carlo).
- `requirements.txt`: Python dependencies.
- `data/`: (Optional) Store training logs, results, or saved models.
- `plots/`: (Optional) Store generated figures.

## Usage
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the main script:
   ```bash
   python main.py
   ```

## Notes
- Modify `main.py` to change training parameters or agent types.
- Results and figures can be saved in the `data/` and `plots/` folders.
