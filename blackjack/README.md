# Blackjack RL Agents

This folder contains scripts, agents, and results for training and evaluating reinforcement learning agents on the Gymnasium Blackjack-v1 environment.

## Files
- `main.py`: Main script for training and plotting results.
- `Agents.py`: Contains agent classes (Q-Learning, Monte Carlo).
- `utils.py`: Helper functions (e.g., moving averages, plotting).
- `data/`: Store training logs, results, or saved models.
- `plots/`: Store generated figures.

## Usage
1. Install dependencies from the root folder:
   ```bash
   pip install -r ../requirements.txt
   ```
2. Run the main script:
   ```bash
   python main.py
   ```
